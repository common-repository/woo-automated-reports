WooCommerce Automated Reports gives you the best way to automate store order reports to your email daily or on specific time.

== Description ==

Get your automated order reports to your email daily or on specific time.

Plugin is compatible with WooCommerce 3.0+ and is tested on 3.3.1 version. 

== Features of the plugin ==

*   <strong> Free version supports only Order Report as text in Email </strong>
*   Create time based and order-status based order report with a time threshold.
*   You can set the time of automated report on the basis of Daily, Recurrence and Specific time.
*   You can select the Report format whether is it text or pdf (Pro Version).
*   You can choose report data whether the order report will have today's or yesterday's order.
*   You can also choose the order statuses which matches with the orders statuses to the reports.
*   Quick dashboard view of Order reports data in a single page for the rules.

For complete list of settings documentation, please visit <a rel="nofollow" href="https://stepqueue.com/documentation/woocommerce-automated-reports-setup/">How To Setup WooCommerce Automated Reports?</a>

== Premium version Features ==

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'WooCommerce Automated Reports'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `woo-automated-reports.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `woo-automated-reports.zip`
2. Extract the `woo-automated-reports` directory to your computer
3. Upload the `woo-automated-reports` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
